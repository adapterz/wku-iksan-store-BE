# 배포 관련 파일 재작성 필요 목록

기존 통합 저장소(`wku-2026-2-kakao-shop`)에서 FE/BE/인프라 저장소로 분리하면서, 아래 배포 관련 파일들은 이 저장소(BE-repo)로 아직 이관되지 않았습니다.
기존 파일들은 FE+BE가 하나의 프로세스로 배포되던 통합 버전 기준이라 그대로 가져와 쓸 수 없고, BE 단독 배포 기준으로 새로 작성해야 합니다.
