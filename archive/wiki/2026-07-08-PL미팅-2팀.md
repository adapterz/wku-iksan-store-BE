# 🗄️ DB 구축 절차

```text
서버 DB 접속
    ↓
DBMS로 서버 DB 연결
    ↓
ERD 기반 SQL 실행
    ↓
테이블 생성 확인
    ↓
시드 데이터 입력
```

## 진행 방식

- 팀원이 각각 DB를 생성하는 방식이 아님.
- 서버에 구축된 공용 DB를 사용하여 작업 진행.
- 서버 DB에 접속한 뒤 ERD 기반 SQL을 실행하여 테이블 생성.
- DB 구축 완료 후 해당 DB를 기준으로 개발 진행.
