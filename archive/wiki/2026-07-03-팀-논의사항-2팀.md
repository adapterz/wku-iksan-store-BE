# 2. 구현 대상 페이지 검토

## 논의 내용

페이지 누락 여부를 확인하기 위해 Figma 디자인 기준으로 전체 화면을 검토하였다.

### 확인 사항

- Figma는 원본이 아닌 **Design 화면 기준**으로 확인
- 누락된 페이지가 있는지 의견 수렴
- 의견이 없는 경우에도 확인 여부를 공유하도록 요청

---

# 3. "선물함 사용하기" 페이지 추가 여부

## 논의

### 제안

- 기존 Wiki에 **"선물함 사용하기"** 페이지가 누락되어 있는 것으로 판단
- 페이지 추가 여부 논의 진행

### 기능 정의

- 결제 직후 사용하는 화면이 아니라
- 선물함에 보관된 상품을 사용할 때
- **바코드를 표시하는 화면**으로 정의

### 의견

- FE에서 구현하는 화면으로 추가하면 좋을 것 같다는 의견
- 구현 방식에 따라 API가 필요할 수도 있다는 의견
- 현재 단계에서는 API 설계보다 **페이지 선별을 우선** 진행하기로 함
